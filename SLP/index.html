ABOUT THE EVENT

If we listen, we forget. If we see, we remember. If we experience, we understand.

If we involve, we explore.

After all, what is the meaning of being a student, if we don't explore. Student leadership program is one such platform where each member gets a chance to show off his skills and talent. Leadership is a trait that is valuable asset each walk of life. It is a very important asset that each person should make use of. Having that thought process imbibed in us let's up open up several doors of knowledge. So come and be part of this... There is going to be fun, learning, masti, joy, drama, playfulness and what not.

"If your actions inspire others to dream more, learn more, do more and become more, you are a leader."
